# Abdulrahman Nahhas

I'm Abdulrahman Nahhas, a 16-year-old programmer from Syria. I'm passionate about technology and I'm always looking for new ways to learn and grow.

I've been programming for more than 3 years now, and I'm proficient in a variety of languages and frameworks, including:

* **Web development:**
  - HTML
  - CSS
  - Bootstrap
  - JavaScript
  - Node.js
  - Express.js
  - React.js
  - Tailwind css
  - Next.js
  - Shadcn/ui
  - and more...
* **Electronics:**
  - electronics basics (some components, circuits, and laws)
  - C++
  - Arduino
  - ESP32 ESP-IDF (currently learning)

I'm currently in grade 11, and I'm planning to study mechatronics engineering in college. I'm excited to continue learning and growing, and I'm confident that I can make a difference in the world with my skills.
